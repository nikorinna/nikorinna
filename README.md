## 🚀 Current project

### FrenchUp, B2+ Booster for TCF Canada

**FrenchUp** is an interactive single-page web app designed to support French learners preparing for **TCF Canada**.

It helps learners strengthen written production through:

* ✍️ B1 to B2+ vocabulary upgrades
* 🔗 Connector practice
* 🧩 Sentence building
* 🔁 Rewriting drills
* 💬 Synonym practice
* 📖 Reading and idea development
* 🧠 Task 3 argument support
* 🎯 Exam-oriented writing strategies

**Tech stack:** HTML, CSS, JavaScript
