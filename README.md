## 🚀 Current project

### FrenchUp, B2+ Booster for TCF Canada

![HTML](https://img.shields.io/badge/HTML-E34F26?style=flat-square\&logo=html5\&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat-square\&logo=css3\&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square\&logo=javascript\&logoColor=black)

**FrenchUp** is an interactive single-page web app designed to support French learners preparing for **TCF Canada**.

It helps learners strengthen written production through:

* ✍️ B1 to B2+ vocabulary upgrades
* 🔗 Connector practice
* 🧩 Sentence building
* 🔁 Rewriting drills
* 💬 Synonym practice
* 📖 Reading and idea development
* 🧠 Task 3 argument support
* 🎯 Exam-oriented writing strategies
